Lenguajes de programación

Lenguaje de programación Año de creación Paradigma Aplicación
C 1972 Estructurado, imperativo. Programación de sistemas, desarrollo de juegos, sistemas embebidos.
C++ 1985 Extensión de C, orientado a objetos. Programación de sistemas, desarrollo de juegos, aplicaciones de alto rendimiento.
Java 1995 Orientado a objetos, imperativo. Desarrollo de aplicaciones web, móviles y de escritorio.
Python 1991 Interpretado, imperativo, orientado a objetos. Desarrollo web, machine learning, cálculo científico, automatización.
JavaScript 1995 Interpretado, funcional. Desarrollo web, elementos interactivos en páginas.
Swift 2014 Orientado a objetos, funcional. Desarrollo de aplicaciones iOS y macOS.
