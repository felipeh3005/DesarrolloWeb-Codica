HTML

HTML (HyperText Markup Language) es un lenguaje para la estructuración de contenido en la web. Es un conjunto de reglas que permite al navegador distinguir entre encabezados y listas, tablas e imágenes, entre otros elementos. HTML se introdujo en 1993 y tenía como objetivo estandarizar las reglas para mostrar texto dentro de las páginas web.

Observa detenidamente la página en la que te encuentras ahora mismo. Independientemente de su apariencia, está descrita completamente en HTML. Esto demuestra que, sin importar lo compleja que sea una página, su esencia se mantiene inalterada durante más de 20 años.

La palabra "lenguaje" en HTML debe entenderse como reglas. HTML solo marca los datos, pero no interactúa con ellos ni los muestra visualmente. Todo el trabajo de mostrar datos de texto y estructurarlos lo realiza el navegador.

CSS

CSS (Cascading Style Sheets, "hojas de estilo en cascada") es un lenguaje formal para describir la apariencia de un documento escrito en un lenguaje de marcado (generalmente HTML o XHTML). También se puede aplicar a cualquier documento XML, como SVG o XUL.

JavaScript

JavaScript es un lenguaje de programación multiparadigma que admite estilos de programación orientados a objetos, imperativos y funcionales. Es una implementación del estándar ECMAScript (estándar ECMA-262).

JavaScript se utiliza normalmente como un lenguaje incrustado para acceder a objetos de aplicaciones. Se utiliza ampliamente en los navegadores como un lenguaje de secuencias de comandos para hacer que las páginas web sean interactivas.

Características arquitectónicas principales: tipado dinámico, tipado débil, gestión automática de memoria, programación basada en prototipos, funciones como objetos de primera clase.

JavaScript ha sido influenciado por muchos lenguajes, y durante su desarrollo se pretendía que fuera similar a Java. JavaScript no es propiedad de ninguna empresa u organización, lo que lo diferencia de muchos otros lenguajes de programación utilizados en el desarrollo web.

El nombre "JavaScript" es una marca registrada de Oracle Corporation en los Estados Unidos.
